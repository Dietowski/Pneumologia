### **Olá amantes da medicina!**

**O que esse projeto faz?**

Já imaginou ter uma enciclopédia de doenças pulmonares na palma da sua mão?  Com essa aplicação, você pode pesquisar por qualquer doença respiratória e encontrar informações importantes sobre sintomas, causas, tratamentos e muito mais! É como ter um consultório médico no seu navegador. 

**Como funciona a mágica?**

1. **Digite a doença:** Na barra de pesquisa, você digita o nome da doença que te interessa.
2. **Resultados instantâneos:** A aplicação vasculha um banco de dados gigantesco (na verdade, um simples array JavaScript no nosso caso) e te mostra todas as informações relevantes sobre a doença pesquisada.
3. **Informações completas:** Para cada doença, você encontra:
   * **Nome:** O nome bonitinho da doença.
   * **Descrição:** Uma breve explicação sobre o que é a doença.
   * **Sintomas:** Os principais sintomas para você ficar de olho.
   * **Causas:** O que pode causar essa doença.
   * **Tratamentos:** Como combater a doença.
   * **Fatores de risco:** Quem tem mais chances de pegar essa doença.
   * **Link para mais informações:** Um botão mágico que te leva para fontes confiáveis para saber mais.

**Tecnologias usadas:**

* **HTML:** A estrutura básica da página, como se fosse os ossos de um esqueleto.
* **CSS:** A roupa bonita da página, deixando tudo organizado e visualmente agradável.
* **JavaScript:** O cérebro da aplicação, responsável por toda a lógica e interação com o usuário.

**Como usar esse projeto:**

1. **Clone o repositório:** Use o git clone para baixar todos os arquivos para o seu computador.
2. **Abra o index.html:** Dê um duplo clique no arquivo index.html para abrir a página no seu navegador.
3. **Comece a pesquisar:** Digite o nome da doença e veja a magia acontecer!

**O que falta para ficar perfeito?**

* **Banco de dados:** No momento, usamos um array JavaScript simples para armazenar as informações das doenças. Poderíamos usar um banco de dados real para ter uma quantidade maior de dados e funcionalidades mais avançadas.
* **Design:** A parte visual da aplicação poderia ser melhorada com um design mais moderno e intuitivo.
* **Funcionalidades extras:** Podemos adicionar filtros, gráficos e outras funcionalidades para tornar a aplicação ainda mais completa.
